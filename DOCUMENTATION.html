FlowPilot — SaaS Landing Page

Project Documentation | Version 1.0 | July 2025

📋 Project Overview

FlowPilot is a modern, high-performance SaaS landing page built with Next.js 16, React 19, Framer Motion, and Tailwind CSS. The project showcases an AI-powered project management platform designed for high-velocity teams.

Key Statistics:

  • 50,000+ teams using the platform
  • 99.9% uptime guarantee
  • 5M+ tasks completed
  • 120+ countries supported
  • 100+ integrations available

🎯 Core Features

AI Assistant
Automatic task generation, meeting summaries, deadline suggestions
Kanban Boards
Drag-and-drop interface with custom workflows and labels
Team Collaboration
Real-time comments, mentions, file uploads, presence indicators
Time Tracking
Built-in timers, productivity reports, billable hours
Analytics
Velocity metrics, burndown charts, team performance
Smart Automation
Recurring tasks, triggers, approvals, notifications

📁 Project Structure

Key directories:

  • app/ - Next.js app router files (layout, page, styles)
  • components/ - Reusable React components organized by type
  • components/sections/ - 12 page sections (Hero, Features, Pricing, etc.)
  • components/ui/ - UI components (Button, Accordion, Logos, etc.)
  • constants/ - Content data and configuration
  • hooks/ - Custom React hooks (useTheme for dark mode)
  • public/ - Static assets

🎨 Page Sections (12 Total)

  1. Navbar - Navigation with logo and CTA
  2. Hero - Main headline with value proposition
  3. Trust Section - Featured companies using FlowPilot
  4. Features - 6 core features with icons
  5. Showcase - Product demonstration
  6. Benefits - 8 key benefits
  7. How It Works - 3-step onboarding process
  8. Integrations - 12 partner tool logos in marquee
  9. AI Section - AI capabilities highlight
  10. Testimonials - 6 customer testimonials with ratings
  11. Pricing - 3 pricing tiers (Starter, Pro, Enterprise)
  12. FAQ - 8 FAQs in accordion format
  13. Final CTA - Bottom conversion funnel
  14. Footer - Links and company info

✨ Design Features

  • Dark/Light theme with system preference detection
  • Smooth scroll animations with Framer Motion
  • Mobile-first responsive design (Tailwind CSS)
  • Semantic HTML for accessibility
  • Optimized font loading (Inter, Plus Jakarta Sans)
  • Real brand logos for 12 integrations (SVG)
  • Marquee carousel for scrolling integrations
  • Lazy loading and code splitting

🚀 Quick Start

Installation

Development

Linting

npm run lint

🌍 Deployment to Vercel

Method 1: GitHub Integration (Recommended)

  1. Push code to GitHub
  2. Visit vercel.com/new
  3. Select your repository
  4. Click "Deploy" (auto-detects settings)
  5. Get live URL in 2-3 minutes

Method 2: Vercel CLI

npm install -g vercel vercel login vercel --prod

Configuration Files

  • vercel.json - Build settings, headers, security
  • .vercelignore - Files to exclude from deployment

📊 Content Management

All content is centralized in constants/data.ts:

  • NAV_LINKS - Navigation items
  • FEATURES - Core features with icons and colors

  • PRICING - Subscription tiers and features

🔧 Customization Guide

Update Content

Edit constants/data.ts for all copy, features, pricing, testimonials

Change Colors

Update Tailwind config or inline style props in components

Modify Fonts

Change font imports in app/layout.tsx

Add Sections

Create component in components/sections/ and import in app/page.tsx
